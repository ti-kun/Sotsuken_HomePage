卒研案内サイト及び、グループページを作成しました。

・フォルダ構成について
拡張子別に各ファイルを格納してあります。
依存関係が崩れてしまうので、展開後そのまま起動してください。

・使い方について
案内サイトは、DBを使用しているので、ファイルリンク直打ちだと正しく機能しません。
以下、手順です。

①"htdocs/Sotsuken_Homepage/"になるようにフォルダを展開してください。

②xamppを起動後、mysqlに"sql/group_list_db.sql"をインポートしてください。

③"php/db.php"の6行目に、実行環境のphpmyadminのログインパスワードを入力してください。
